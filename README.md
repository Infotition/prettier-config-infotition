# Installation

1. Install the npm package:
   ```bash
   npm i -D @infotition/prettier-config
   ```
   or with yarn:
   ```bash
   yarn add -D @infotition/prettier-config
   ```

2. Create add the following entry to the root level `package.json` file:
   ```json
   {
     "prettier": "@infotition/prettier-config"
   }
   ```
